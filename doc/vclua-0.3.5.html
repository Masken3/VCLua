VCLua version0.3.5

Library functions

Getting desktop size

width, height = VCL.GetScreenSize()

Getting mouse position

x, y = VCL.GetCursorPos()

Displaying message in popup window

VCL.ShowMessage(string)

Displaying message dialog in popup window

string = VCL.MessageDlg(string,string,string or table)

    example:

         if VCL.MessageDlg( "You have unsaved file(s). Continue?" , "mtConfirmation" , { "mbYes" , "mbCancel" } ) == "mrYes" then ...

Checking of file existence

boolean = VCL.FileExists(string)

Checking of directory existence

boolean = VCL.DirectoryExists(string)

Listing properties of a vclua object

table = VCL.ListProperties(object)

Running lua script in separated environment

result = VCL.RunSeparate(number, script, {params})

example:

result = VCL.RunSeparate(table.maxn(scriptArgs),scriptName,unpack(scriptArgs))
